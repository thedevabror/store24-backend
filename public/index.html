API Documentation

This documentation provides an overview of the endpoints and functionality of our application.

Authentication

POST /api/admin/register

Create a new admin user.

{
  "name": "Admin Name",
  "email": "admin@example.com",
  "password": "password"
}

POST /api/admin/login

Login as an admin user.

{
  "email": "admin@example.com",
  "password": "password"
}

Products

POST /api/products

Create a new product.

{
  "name": "Product Name",
  "description": "Product Description",
  "price": 100,
  "countInStock": 10,
  "category": "Category ID",
  "brand": "Brand",
  "color": "Color",
  "attributes": "{\"key\":\"value\"}"
}

GET /api/products

Get all products.

GET /api/products/:id

Get a product by ID.

PUT /api/products/:id

Update a product by ID.

{
  "name": "Updated Product Name",
  "description": "Updated Product Description",
  "price": 150,
  "countInStock": 20,
  "category": "Category ID",
  "brand": "Updated Brand",
  "color": "Updated Color",
  "attributes": "{\"key\":\"updated value\"}"
}

DELETE /api/products/:id

Delete a product by ID.

GET /api/products/top

Get top-selling products.

Orders

POST /api/orders

Create a new order.

{
  "userId": "User ID",
  "products": [
    {
      "productId": "Product ID",
      "quantity": 2
    }
  ],
  "totalPrice": 200
}

GET /api/orders

Get all orders (Admin only).

GET /api/orders/:id

Get an order by ID.

PUT /api/orders/:id

Update order status by ID.

{
  "status": "completed" // or "cancelled"
}

DELETE /api/orders/:id

Delete an order by ID.

Categories

POST /api/categories

Create a new category.

{
  "name": "Category Name",
  "parent": "Parent Category ID" // optional
}

GET /api/categories

Get all categories.

GET /api/categories/:id

Get a category by ID.

DELETE /api/categories/:id

Delete a category by ID.

Cart

POST /api/cart

Add a product to the cart.

{
  "userId": "User ID",
  "productId": "Product ID",
  "quantity": 1
}

GET /api/cart/:userId

Get the cart for a user by user ID.

DELETE /api/cart

Remove a product from the cart.

{
  "userId": "User ID",
  "productId": "Product ID"
}

DELETE /api/cart/clear/:userId

Clear the cart for a user by user ID.

Users

GET /api/users

Get all users (Admin only).

PUT /api/users/block/:id

Block or unblock a user by ID.

{
  "isActive": false // or true
}
